In Valid Communications

You were born during the 5th great war between the planets Earth and Zopita. Growing up, you despised the atrocities of interplanetary conflict and dreamt of universal peace. After years of warfare, it all boiled down to you solving a computational problem to put the wretched war to an end! And this was what you had to do:

Given an unbroken stream of characters, transmitted from the Zopita Headquarters, you had to determine whether the stream was a string (a sentence) of valid English words, or not. For example, the stream “uuydhllhfis” does not give a string of valid words, but the stream, “unleashdeadlyturtleheadsonearthplease”, is just the sentence: “unleash deadly turtleheads on earth please”. To your aid, you have a lexicon of concise 40th century English, and given a string str, this lexicon returns true if str is a valid English word and false otherwise, in constant time.

Write a program to solve this problem. The dictionary is a list of strings provided with this statement. The function bool validWord(std::string &str) returns true if str is a valid word in the dictionary and false otherwise.

Stage 1:

C++ Function Header:

bool validCommunication(const std::string& stream);

If a solution is possible, the function returns true; otherwise, it returns false.

Stage 2:

C++ Function Header:

bool validCommunication(const std::string& stream, std::string & decoded);

If a solution is possible, the function returns true and stores into decoded the sentence containing space-separated valid words; otherwise, the function returns false and sets decoded to an empty string.

Dictionary: Use the 10,000 words list available here: https://www.mit.edu/~ecprice/wordlist.10000.
