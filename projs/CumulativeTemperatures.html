Problem 1.2 Cumulative Temperatures

As a data analyst for an environmental research company, you need to identify periods of extreme temperature fluctuations from sensor data collected over time. Your task is to find the contiguous stretch of recorded temperature values that has the highest cumulative temperature. The data includes both positive and negative values, indicating periods of warming and cooling. By identifying the stretch with the maximum sum, you can pinpoint the most significant period of temperature change for further analysis.

Given a sequence of temperature readings, determine the start and end indexes of the contiguous subsequence that yields the maximum sum, along with the maximum cumulative value itself.

Examples:

  1. Input: temperatures = [2, -4, 3, -1, 5, -2, 7, -3]

    Output: start_index = 2, end_index = 6, max_sum = 12

    Explanation: The contiguous stretch with the maximum sum is [3, -1, 5, -2, 7], starting at index 2 and ending at index 6. Hence, the maximum cumulative value is 12.

  2. Input: temperatures = [-5, -1, -8, 4, -2, 6, -3, 1]

    Output: start_index = 3, end_index = 5, max_sum = 8

    Explanation: The contiguous stretch with the maximum sum is [4, -2, 6], starting at index 3 and ending at index 5. Hence, the maximum cumulative value is 8.

  3. Input: temperatures = [1, -3, 2, 1, -1, 3, -2, 4, -5]

    Output: start_index = 2, end_index = 7, max_sum = 7

    Explanation: The contiguous stretch with the maximum sum is [2, 1, -1, 3, -2, 4], starting at index 2 and ending at index 7. Hence, the maximum cumulative value is 7.
