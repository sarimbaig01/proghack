Shuffle Playlist

We often need to shuffle the songs in the playlist, in other words, randomize their order. For a list of n songs, with n! possible orders, an ideal shuffle would ensure that each order occurs with the probability 1/n!. There are several ways of doing this; for example, one way would be to generate a random key corresponding to each song, then sort the songs in order of these keys. This method would require using a sorting method.

However, this can be accomplished faster, in time linear on the size of the input, by using other shuffling methods, such as the Fisher-Yates shuffle. Implement the Fisher-Yates method to shuffle the playlist.

The prototype of the function to be written is:

void shufflePlaylist(std::vector& data, std::vector& links, int& start);
