Sparse Polynomials

Introduction

A sparse polynomial is a polynomial in which most of the coefficients are zero, with only a few non-zero terms. For example, the polynomial \( 5x^8 + 3x^2 + 7 \) is sparse because it has only three non-zero terms, while the remaining coefficients are zero. Another example is \( 4x^{100} + 6x^5 + 2 \), where the highest power is much larger, but the number of non-zero terms remains small.

For non-sparse polynomials, a vector is a suitable data structure to store coefficients, with the index numbers corresponding to the powers of \(x\). For example, the polynomial \(3x^3 + 2x^2 + x + 4\) can be stored as \([4, 1, 2, 3]\), where the value at each index represents the coefficient of the corresponding power of \(x\). The advantage of using a vector is that it allows direct access to any coefficient in constant time, making operations like evaluation or differentiation straightforward and efficient. However, since most coefficients in sparse polynomials are zero, storing all terms in a vector would be wasteful in terms of memory. A more space-efficient approach is to store only the non-zero coefficients. A suitable data structure for this scenario is a linked list, where each node stores a non-zero coefficient and its corresponding power of \(x\), allowing memory to be used only for meaningful terms.
