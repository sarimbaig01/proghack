Problem 1.1 Mirror with Gaps

Consider a list of integer values representing an image (usually this would be a two-deimensional grid but for this problem we consider only one dimension). The image is to be reflected by a peculiar kind of mirror: some parts of the mirror are frozen, i.e, these parts are not reversed but are frozen in their places. This means that when the image is mirrored, all elements in the list are reversed in order except the elements at the positions corresponding to the frozen parts. The reversal of elements happens around the frozen indices.

An additional challenge is that to achieve this peculiar mirrored effect, you should reverse the contents of the list in-place, i.e., not using a second list in the process. The frozen indices are provided separately in the input.

Examples of inputs and outputs:

Two example inputs and their corresponding outputs are shown below.

Input: image = [5, 3, 2, 6, 4, 0, 11, 9, -7, 13], frozen indices = [0, 2, 4, 6]

Output: [5, 13, 2, -7, 4, 9, 11, 0, 6, 3]

Explanation: All elements except the ones appearing on frozen indices (shown in red here) have been reversed in order.
